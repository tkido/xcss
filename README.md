## Usage
-c string
      short form of "class"
-class string
      classes apply to all elements. separator is space e.g. "foo bar"
-d    short form of "debug"
-debug
      add comment where attribute's value came from
-r string
      short form of "root" (default ".")
-root string
      path to start recursive walk (default ".")
